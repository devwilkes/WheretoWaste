# WheretoWaste
A personal project of mine; WheretoWaste will be an educational website that allows users to learn how to dispose of waste products properly, and where they can do so.

## Finding Solutions for Waste Products
People may find themselves with items needing disposal, but have no idea how or where to properly dispose of them. Things like electronic devices, clothing, chemicals, bulk items, and more can often be difficult to figure out, and web resources can only provide so much information on how to properly dispose of them. I decided to create a web solution to this issue because I also found myself struggling to find resources on where and how I could dispose of an old laptop without tossing it directly into the garbage.

## Sustainable Solutions as a Main Focus
WheretoWaste will be focused on providing disposal solutions that are sustainable or provide back to communities. Search results in WheretoWaste for recyclable items will reflect the best possible ways to prepare and recycle those items, for example.

## Built with Python Django Web Framework
Using skills I picked up during my time as an information science major at the University of Maryland, I will use the Python Django web framework integration to build the backend of the website.
